# Analyser les Résultats

La page "Résultats Globaux" est un tableau de bord interactif puissant qui vous permet d'explorer les retours que vous avez reçus.

## Accéder à la Page des Résultats

Depuis le tableau de bord principal de l'administrateur, cliquez sur la carte **"Résultats Globaux"**.

## Filtrer vos Données

En haut de la page, vous trouverez deux barres de filtres pour affiner les données que vous consultez.

### Filtres Principaux

- **Filtre par Campagne** : Un menu déroulant pour sélectionner une campagne spécifique. Par défaut, il affiche "Toutes les campagnes". Lorsque vous sélectionnez une campagne, le filtre des managers se met à jour pour n'afficher que les managers de cette campagne.
- **Filtre par Manager** : Un menu déroulant pour filtrer les résultats d'un manager ou d'une équipe spécifique. Par défaut, il affiche "Tous les managers".

### Filtre par Période

Sous les filtres principaux, vous pouvez sélectionner une plage de temps pour les données :

- **7 jours**, **30 jours**, **3 mois**, **Année en cours**, ou **Tout**.
- La vue par défaut est "Année en cours". Les données du tableau de bord se mettront automatiquement à jour lorsque vous sélectionnerez une nouvelle période.

## Comprendre le Tableau de Bord

Le tableau de bord est composé de plusieurs widgets interactifs qui vous donnent une vue complète de l'humeur de vos équipes.

### 1. Graphique de Répartition des Humeurs

- **Ce que c'est** : Un graphique circulaire montrant le pourcentage de chaque humeur (Très bien, Neutre, etc.) pour les filtres sélectionnés.
- **Interactivité** : Survolez une section du graphique avec votre souris. L'arrière-plan de toute la page changera en douceur pour prendre la couleur de cette humeur, offrant une expérience visuelle dynamique.

### 2. Derniers Commentaires

- **Ce que c'est** : Une liste déroulante de tous les commentaires laissés par les utilisateurs, correspondant à la sélection de filtres actuelle.
- **Détails** : Chaque commentaire est anonyme mais indique de quel manager/équipe provient le retour, accompagné d'un point de couleur indiquant l'humeur associée au commentaire.

### 3. Cartes de Statistiques Clés

Sous le graphique et les commentaires, deux cartes mettent en évidence les métriques les plus importantes :

- **Votes Totaux** : Le nombre total de votes reçus pour la sélection actuelle.
- **Humeur Dominante** : L'humeur qui a reçu le plus de votes.
- **Interactivité** : Survolez la carte "Humeur Dominante". L'arrière-plan de la page changera pour prendre la couleur de cette humeur, tout comme avec le graphique.

## Exporter les Données

Vous pouvez exporter les données filtrées à tout moment.

- Cliquez sur le bouton **"Exporter (CSV)"** situé dans la barre de filtres principale.
- Un fichier CSV sera téléchargé sur votre ordinateur.
- Le nom du fichier sera automatiquement généré en fonction de la campagne sélectionnée et de la date actuelle (par ex., `resultats_Sprint-23_2025-11-03_15-00.csv`).
- Ce fichier contient un détail de tous les votes de la sélection actuelle, y compris la date, la campagne, le manager, l'humeur et le texte complet du commentaire.
