# Gérer les Campagnes

Une fois que vous avez créé une ou plusieurs campagnes, la page "Campagnes Actives" devient votre centre de contrôle pour le suivi et la gestion.

## Accéder à la Page de Gestion

Depuis le tableau de bord principal de l'administrateur, cliquez sur la carte **"Campagnes Actives"**.

## Le Tableau des Campagnes

Vous verrez un tableau listant toutes vos campagnes. Par défaut, il n'affiche que les campagnes actives (non archivées). Le tableau fournit les informations suivantes en un coup d'œil :

- **Nom** : Le nom de la campagne.
- **Liens** : Le nombre total de liens de sondage uniques générés pour cette campagne.
- **Votes** : Le nombre total de votes reçus pour cette campagne jusqu'à présent.
- **Date** : La date de création de la campagne.
- **Participation** : Une barre de progression indiquant le pourcentage de liens qui ont reçu au moins un vote.

### Afficher les Campagnes Archivées

Pour voir les campagnes que vous avez précédemment archivées, il suffit d'activer l'interrupteur **"Afficher les archivées"** en haut à droite de la page. Le tableau se mettra à jour pour n'afficher que les campagnes archivées, et le menu d'actions changera en conséquence.

## Actions sur les Campagnes

Pour chaque campagne dans le tableau, il y a un menu déroulant **"Actions"** tout à droite. Ce menu offre toutes les options de gestion pour cette campagne spécifique.

### 1. Gérer les liens

- **Action** : "Gérer les liens"
- **Ce que ça fait** : Ouvre une boîte de dialogue qui liste tous les liens de sondage associés à cette campagne.
- **Fonctionnalités** :
  - **Copier le lien** : Vous pouvez copier n'importe quel lien individuel dans votre presse-papiers en utilisant l'icône de copie.
  - **Ajouter un manager** : Cliquez sur le bouton **"Ajouter un manager"** en bas de la fenêtre. Une nouvelle fenêtre apparaîtra, vous permettant de saisir le nom d'un nouveau manager. Un nouveau lien unique sera généré pour lui et ajouté à la liste.

### 2. Archiver / Restaurer

- **Action** : "Archiver" ou "Restaurer"
- **Ce que ça fait** : Cette action change selon que vous consultez les campagnes actives ou archivées.
  - **Archiver** : Pour une campagne active, cela la déplacera vers les archives. Elle ne sera plus visible dans la vue par défaut.
  - **Restaurer** : Pour une campagne archivée (lorsque "Afficher les archivées" est activé), cela la rendra à nouveau active.

### 3. Voir les résultats

- **Action** : "Voir les résultats"
- **Ce que ça fait** : C'est un raccourci qui vous amène directement à la page "Résultats Globaux", avec le filtre déjà appliqué pour n'afficher que les résultats de cette campagne spécifique.
