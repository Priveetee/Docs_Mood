# Créer une Campagne

La création d'une campagne est l'action principale pour commencer à recueillir des retours. Ce guide couvre le processus du début à la fin.

## Accéder à la Page de Création

Depuis le tableau de bord principal de l'administrateur, cliquez sur la carte **"Nouvelle Campagne"** pour accéder à la page de création.

## Étape 1 : Remplir les Détails de la Campagne

Un formulaire vous est présenté pour configurer votre nouvelle campagne.

1.  **Nom de la campagne** : Saisissez un nom clair et descriptif pour votre campagne (par exemple, "Feedback Fin d'Année", "Rétrospective Sprint 23"). Ce nom sera visible pour vous dans le tableau de bord et pour les utilisateurs sur la page de sondage.

2.  **Ajouter des managers** : C'est ici que vous listez les équipes ou les managers pour lesquels vous souhaitez générer des liens de sondage uniques.
    - Tapez le nom d'un manager (ou d'une équipe) dans le champ de saisie.
    - Cliquez sur le bouton **"+"** ou appuyez sur **Entrée**.
    - Le nom apparaîtra sous forme de badge sous le champ. Vous pouvez supprimer un manager en cliquant sur le "x" de son badge.
    - Vous devez ajouter au moins un manager pour continuer.

3.  **Date d'expiration (Optionnel)** :
    - Cliquez sur le bouton du calendrier pour ouvrir un sélecteur de date.
    - Sélectionnez une date future à laquelle vous souhaitez que la campagne se termine automatiquement. Après cette date, plus aucun vote ne sera accepté.
    - Si vous ne définissez pas de date, la campagne restera ouverte indéfiniment jusqu'à ce que vous l'archiviez manuellement.

## Étape 2 : Générer les Liens

Une fois que vous avez fourni un nom et ajouté au moins un manager, le bouton **"Générer les liens"** deviendra actif.

- Cliquez sur ce bouton pour créer la campagne et générer un lien de sondage unique et anonyme pour chaque manager que vous avez ajouté.

## Étape 3 : Distribuer les Liens

Après la génération, l'affichage basculera sur un écran de résultats montrant les liens fraîchement créés.

- **Liste des liens** : Vous verrez une liste de chaque manager avec son URL unique correspondante.
- **Copier un lien individuel** : Utilisez l'icône de copie à côté de chaque lien pour le copier individuellement.
- **Copier tout** : Cliquez sur le bouton **"Copier tout"** pour copier la liste entière des managers et de leurs liens dans votre presse-papiers, prête à être collée dans un e-mail ou un document.
- **Envoyer par email** : Cliquez sur le bouton **"Envoyer par email"**. Cela ouvrira votre client de messagerie par défaut avec un brouillon pré-rempli contenant le nom de la campagne et la liste de tous les liens générés, prêt à être envoyé.

Vous pouvez maintenant distribuer ces liens aux personnes appropriées. Une fois que les votes commenceront à arriver, vous pourrez les suivre en gérant la campagne ou en analysant les résultats.
